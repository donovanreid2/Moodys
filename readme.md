4. Azure Kubernetes Managed Namespace
Set resource limits within namespace, network policies, and grant team access via RBAC, also labels specifying who the namespace is for. Create managed Namespaces using for each loop. For example namespace for finace, HR, Legal, etc.
Managed Namespace built in roles:
Azure Kubernetes Service Namespace Contributor
Azure Kubernetes Service Namespace user
Azure Kubernetes service rbac reader
Azure kubnetes service RBAC writer
Azure Kubernetes service rbac admin
